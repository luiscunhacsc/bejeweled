Projeto Bejeweled Didático em Java (Swing)
===========================================

Este projeto implementa um jogo simples do tipo Bejeweled, adequado para estudantes de Introdução à Programação.

Como executar:
--------------
1. Certifique-se de que as imagens das gemas estão na pasta 'gemsPNG' dentro da raiz do projeto.
2. Compile todos os ficheiros Java:
   javac *.java
3. Execute o jogo:
   java Main

Estrutura dos ficheiros:
-----------------------
- Main.java        : Classe principal para iniciar o jogo.
- GameBoard.java   : Painel e lógica principal do jogo.
- Gem.java         : Representa uma gema individual.
- GemType.java     : Enum para os tipos de gemas.
- gemsPNG/         : Pasta com as imagens das gemas.

Notas pedagógicas:
------------------
- O código está comentado para facilitar o estudo.
- A lógica de detecção de combinações e pontuação pode ser expandida pelos alunos.
- O projeto utiliza apenas bibliotecas padrão do Java.

Bom estudo!
